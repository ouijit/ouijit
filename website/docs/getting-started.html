Getting Started

Install Ouijit and launch your first project in under a minute.

Requirements

  • macOS 13+ (Ventura) or Linux (x64)
  • Git 2.20+
  • A code project with a git repository

Download & Install

Download the latest release for your platform:

  • macOS (Apple Silicon)
  • macOS (Intel)
  • Linux (x64)

Unzip the archive and move Ouijit.app to your Applications folder (macOS) or extract the binary to a location on your PATH (Linux).

Building from source

Requires Node.js 20+ and C/C++ build tools for native modules (better-sqlite3, node-pty, koffi):

  • macOS: xcode-select --install
  • Linux: sudo apt install build-essential python3
git clone https://github.com/ouijit/ouijit.git
cd ouijit
npm install
npm start

First Launch

When you first open Ouijit you'll see the project grid — an empty canvas ready for your repositories.

Adding a project

  1. Click the + button or press Cmd+N (Ctrl+N on Linux).
  2. Select a folder containing a git repository.
  3. Ouijit scans the project for language, description, and sets up the database.

Opening a project

Click any project card to enter project mode. You'll see the kanban board on the left and terminal card stack on the right.
