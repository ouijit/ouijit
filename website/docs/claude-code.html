Claude Code Integration

Live status lights so you know which agents are thinking and which are waiting. I boiled several oceans to make these little dots change color.

What you get

When Claude Code is running in an Ouijit terminal, the terminal card shows a live status indicator — thinking or idle. That's it. It's a colored dot. You'd be surprised how useful it is when you have 5+ agents running(?) at once.

How it works

Ouijit installs a thin wrapper that shadows the claude command inside its terminals. When you launch Claude Code, the wrapper automatically injects hook settings that report status back to Ouijit. No configuration needed on your end — just type claude and it's wired up.

Notifications

When an agent finishes, you'll hear a sound. If the window isn't focused, you also get an OS notification. Alt-tab guilt-free while Claude does the work.

What about other agents?

Ouijit's terminals work with any CLI agent — Cursor, Aider, Codex, whatever. They just won't get the fancy status lights. Yet. I play favorites for now.

If your favorite agent has a hook or plugin system, open an issue and I'll see what I can do.
