Kanban Board

A four-column board for organizing tasks, with drag-and-drop and keyboard shortcuts.

Columns

The board has four fixed columns that map to a task's lifecycle:

ColumnMeaning
To DoPlanned work. No worktree or branch created yet.
In ProgressActive work. A worktree and branch are created when a task moves here.
In ReviewWork is done, ready for review. Worktree is still active.
DoneCompleted. The branch can be merged and the worktree removed.

Creating tasks

Click the + button on any column header, or press N to create a new task in the To Do column. Give it a title and optional description.

Each task gets an auto-incrementing number (e.g. T-1, T-2) used for branch names and worktree directories.

Drag and drop

Drag task cards between columns to change their status. You can also reorder tasks within a column.

When dragging a task from To Do to In Progress, Ouijit automatically creates a git worktree and branch for the task.

Task actions

Right-click a task card or click the actions menu to access:

  • Start — Move to In Progress and create worktree
  • Edit — Change title or description
  • Delete — Remove the task (and its worktree if active)
  • View diff — Open the diff panel showing changes vs. main

Keyboard shortcuts

ShortcutAction
NNew task
EnterOpen selected task terminal
BackspaceDelete selected task
