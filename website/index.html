Stuff

This combo should be SOTA for a few weeks.

Kanban Board

Manage parallel work and see status at a glance. Task lifecycle transitions trigger hooks, so you can script workflows around key moments like starting or completing work.

Terminal Stack

As many terminals as you need, all running in the task's worktree. Stack a dev server, an agent, and a manual shell.

Worktree Isolation

Every task runs in its own git worktree — automatic branch creation, instant copy-on-write cloning, and clean merges back to main.

VM Sandbox

Terminal sessions can run inside a Linux VM, fully isolated from your host. Agents can't touch what they shouldn't.

Claude Code Integration

Live status indicators show what Claude Code is doing in each terminal. Know at a glance which agents are thinking and which are idle.

Keyboard-Driven

Navigate, create tasks, switch terminals, and manage workflows without leaving the keyboard.
